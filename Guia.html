¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Utiliza etiquetas para estructurar y presentar contenido en la web.

Etiquetas HTML

Las etiquetas en HTML son los elementos básicos que definen la estructura de una página web. Algunas de las más comunes incluyen:

  • <h1> a <h6> - Títulos con diferentes niveles
  • <p> - Párrafos
  • <a> - Enlaces
  • <img> - Imágenes
  • <div> - División de bloques de contenido

Párrafos

Los párrafos en HTML se crean usando la etiqueta <p>. Permiten organizar el contenido en bloques de texto.

<p>Este es un párrafo de ejemplo.</p>

Imágenes

Para insertar una imagen en HTML, utilizamos la etiqueta <img>. Ejemplo:

<img src="https://via.placeholder.com/150" alt="Imagen de ejemplo">

Video

Para insertar videos, se utiliza la etiqueta <video>:

<video controls><source src="ruta/del/video.mp4" type="video/mp4"></video>
